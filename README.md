# AI Student Productivity Dashboard

A modern AI-powered productivity dashboard built using HTML, CSS, and JavaScript.

## Features
- 🎯 Student Dashboard
- 📝 Today's Goals
- ⏱ Stopwatch
- 🔢 Digital Counter
- 🧮 Calculator
- 🎲 Guessing Number Game
- 🌙 Dark/Light Mode
- 👤 Change Profile Picture
- 🖼 Animated Image Slider
- 🔍 Search Bar

## Technologies
- HTML5
- CSS3
- JavaScript
